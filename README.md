安裝beego請參考 [安裝Beego cli](https://ithelp.ithome.com.tw/articles/10192092)

如果你想要使用此程式

- 切換到您的GOPATH 目錄
```bash
$ cd $GOPATH/src
```

- Clone這個branch
```bash
$ git clone git@github.com:StockinTea/ironSchool.git
```

- 執行ironSchool
```bash
$ bee run ironSchool
```
